== Setup of Willy
The software of Willy is packaged inside Docker containers, so every system that is compatible with Docker could run Willy. This setup describes how a developer can run the containers of Willy.

=== Installation of Willy

- link:https://docs.docker.com/install/[Install Docker]
// tag::installation[]
- Clone the source code and configuration of Willy

[source,shell]
----
https://github.com/Windesheim-Willy/willy.git
cd willy
git submodule update --init --recursive
----

- Start Willy

[source,shell]
----
docker-compose up
----

[NOTE]
--
At some Linux systems it is required to run docker and docker-compose with sudo.
--
// end::installation[]

=== Update of Willy

Stop Willy
[source,shell]
----
docker-compose stop
----

Get the new changes from Github
[source,shell]
----
git pull origin master
git submodule update --recursive --init
docker-compose pull
----

Start Willy
[source,shell]
----
docker-compose up
----
