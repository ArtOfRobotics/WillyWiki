== Kinect
The kinect is a node to process the raw kinect information to the kinect topic.

=== Requirements
This component has a requirement to a Kinect. The Kinect should be connected to the host system by the USB port. Docker will mount the kinect in the kinect component container with the link:https://docs.docker.com/compose/compose-file/compose-file-v2/#devices[devices configuration].

https://github.com/Windesheim-Willy/kinect[Windesheim-Willy/kinect, role="external", window="_blank"]

=== How to run?
To run only the kinect the Docker can be used. The Dockerfile for the kinect is automatically built on Dockerhub.

====
https://hub.docker.com/r/windesheimwinny/kinect[windesheimwinny/kinect, role="external", window="_blank"]
====

Run the build docker image:

[source,shell]
----
docker-compose up
----

For development purposes it can be usefull to build the Dockerfile locally and to use this image. The docker-compose.build.yml file builds first the image and after the build start the container.

[source,shell]
----
docker-compose -f docker-compose.yml -f docker-compose.build.yml up
----
